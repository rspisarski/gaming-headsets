## Overview
The Corsair Void Elite combines wireless freedom with immersive 7.1 surround sound, making it an excellent choice for PC and PlayStation gamers. With comfortable memory foam ear cushions and customizable RGB lighting, this headset delivers both performance and style.

## Key Features
- 7.1 surround sound for immersive gaming audio
- 20-hour wireless battery life with fast charging
- Memory foam ear cushions with microfiber mesh
- Discord certified microphone with flip-to-mute functionality
- RGB lighting with Corsair iCUE software customization

## Specifications
The Void Elite features 50mm neodymium drivers with a frequency response of 20Hz-20kHz and 32-ohm impedance. Weighing 374 grams, it offers comfortable extended wear with breathable memory foam ear cushions. The omnidirectional microphone is Discord certified and automatically mutes when flipped up. Battery life reaches 20 hours with RGB lighting enabled.
