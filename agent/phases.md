# Project Phases

## Phase 1: Foundation & Styling Setup
- Set up color variables (Black, White, Purple #6738E7)
- Configure fonts (Poppins, Noto Sans)
- Implement dark theme (background: black, text: white, accents: purple)
- Set up border-radius (.5rem)

## Phase 2: Navigation Bar
- Create sticky navigation with scroll blur effect
- Add text logo "Gaming Headsets"
- Add navigation links (Find your headset, Blog)
- Add search icon (UI only, no functionality)
- Add favorites list icon for local storage
- Layout: logo left / nav center / icons right

## Phase 3: Hero Section
- Design featured blog post (left side - most recent)
- Design blog post list (right side, 1 column)
- List items: 1:1 aspect ratio image, title, excerpt, category, date
- Show 3-5 posts in the list

## Phase 4: Blog Page
- Create blog listing page
- One column post list
- Load more functionality
- Similar list design as hero (no featured post)

## Phase 5: Single Post Page
- Full image at top
- Title, excerpt, category, date, content
- Related posts section (filtered by category)

## Phase 6: Integration & Testing
- Connect all pages together
- Test navigation
- Verify responsive design
- Final testing
